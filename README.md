# Lasesarre Intelligence: AI-Driven Tactical Matching and Recruitment System

Este proyecto desarrolla una plataforma escalable de Inteligencia Artificial aplicada al futbol profesional, tomando como caso de estudio al Barakaldo CF (Primera RFEF). El sistema utiliza algoritmos de Machine Learning para optimizar el scouting de fichajes mediante perfiles de similitud tactica y la deteccion de vulnerabilidades mecanicas en equipos rivales.

## Estructura del Proyecto
* data/: Almacenamiento local de metricas y estadisticas avanzadas (Scraping/APIs).
* src/models/: Algoritmos de Machine Learning (Similitud del Coseno, Clustering).
* app/: Interfaz interactiva para la direccion deportiva desarrollada en Streamlit.

## Tecnologias Utilizadas
* Python (Pandas, Scikit-Learn, Soccerplots)
* Streamlit
